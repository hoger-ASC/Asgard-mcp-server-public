# **"Asgard" MCP server - public FOSS version**


### <u>Overview</u>
A modular web-http based *MCP + webapp(uvicorn/fastAPI) + ollama agent* stack written in python; Frontend/ Javascript written by Claude™5[*]

## <u>Overview of Tooling and General Capabilities</u>
The base tooling content of is bulky and still in development and testing; If using this app, you may simply exclude the
unnecesarry libraries and remove the tool's code; It includes:
- search_files
- fetch_url
- read_text_file
- search_knowledge_base
- embedImage
